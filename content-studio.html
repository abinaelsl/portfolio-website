How this works (read me once)
  1. Connect your site folder: click Connect site folder and pick the portfolio-website folder. (Works in Chrome, Edge, Brave. In Safari/Firefox, use the Import / Download buttons instead.)
  2. Add or edit Writing posts, Projects, and Research below. Changes are auto-saved in this browser as you type.
  3. Click "Save to site" — this writes the JSON files in app/content/.
  4. Publish: in a terminal in the project folder, run git add -A && git commit -m "content update" && git push (or npx vercel --prod). Your live site updates in a minute or two.

Tip: Writing post bodies use Markdown — ## Heading, **bold**, *italic*, - list item, > quote, [link](https://…). To add a project screenshot, drop the image in public/projects/ and put its path (e.g. /projects/myshot.png) in the Image field.
